# 如何搭建本地服务器

## 开启IIS互联网信息服务
```
1.  打开控制面板->程序->程序和功能->启动或关闭Windows功能

2.  选中Internet Information Services，点击确定
```

## 搭建本地服务器
```
1.  控制面板主页，查看方式选为大图标

2.  打开管理工具，打开Internet Information Services

3.  将左侧展开，在“网站”上右键选择“添加网站”

4.  填写“网站名称”
    选择应用程序池设为“DefaultAppPool”
    选择“物理路径”(也就是你服务器的根目录)
    最后点击确定,服务器就建好了

5.  在物理路径下放置index.html,右侧选择浏览网站
```

## 外网访问设置(路由器)
```
1.  浏览器输入路由器背面的id地址，使用路由器默认的用户名和密码登录

2.  选择“转发规则->虚拟服务器->添加新条目”
    输入IP地址(本地IP地址)
    端口号(IIS网站端口号)
    选择协议设为“ALL”(一般为TCP/IP)
    选择状态设为“生效”
    最后点击确定

3.  选择“转发规则->DMZ主机设置”
    选择DMZ状态为“启用”
    输入DMZ主机IP地址(本地IP地址)
    最后点击确定

4.  外网测试(浏览器输入Ip:Port是否能正常访问本地网站)
```
